# Release Notes

## v1.0.0 (2026-03-07)

### 🎉 첫 번째 정식 릴리스

Cash Log는 개인 재무를 간편하게 관리할 수 있는 데스크톱 애플리케이션입니다. Electron과 React 기반으로 제작되었으며, 로컬 데이터베이스를 사용하여 모든 데이터를 안전하게 보관합니다.

### ✨ 주요 기능

#### 📅 수입/지출 관리
- 날짜, 금액, 카테고리, 태그, 메모를 포함한 거래 기록
- 사용자 정의 카테고리 및 태그 시스템
- 기간별, 카테고리별, 태그별 필터링 및 검색

#### 📊 데이터 시각화
- 월별 캘린더 뷰로 일별 수입/지출 확인
- 월별 지출 추이 차트
- 카테고리별 지출 분석 (파이 차트)
- 상위 수입/지출 내역 조회

#### 💾 데이터 관리
- PostgreSQL 기반 로컬 데이터베이스
- CSV, Excel, PDF 형식으로 보고서 생성
- 안전한 데이터 영속성 보장

### 🛠️ 기술 스택
- **Frontend**: Electron, React, TypeScript, Vite
- **Backend**: Spring Boot, Java 21
- **Database**: PostgreSQL
- **Infrastructure**: Docker, Docker Compose

### 📦 설치 방법

#### macOS (권장)
```bash
# 빌드 및 설치
cd apps/frontend
npm run build
npm run package:mac
sudo installer -pkg "release/Cash Log-1.0.0-universal.pkg" -target /

# 앱 실행 (DB와 백엔드 자동 시작)
open -a "Cash Log"
```

**참고**: 설치된 앱은 실행 시 DB와 백엔드를 자동으로 시작합니다.

#### 개발 모드
```bash
# 전체 애플리케이션 시작
./start-app.sh
```
