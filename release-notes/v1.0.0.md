# Release Notes

## v1.0.0 (2026-03-07)

### 🎉 첫 번째 릴리스

Cash Log의 첫 번째 안정 버전을 출시합니다!

### ✨ 주요 기능

#### 💰 수입/지출 관리
- 날짜, 금액, 카테고리, 태그, 메모를 포함한 거래 기록
- 다중 통화 지원 (KRW, USD, EUR, JPY, CNY)
- 카테고리별 분류 및 관리
- 태그 시스템으로 세밀한 분류

#### 📊 데이터 시각화
- 월별 캘린더 뷰
- 카테고리별 지출 분석 (파이 차트)
- 월별 지출 추이 (라인 차트)
- Top 수입/지출 내역

#### 📁 데이터 관리
- CSV, Excel, PDF 보고서 생성
- 로컬 MySQL 데이터베이스 저장
- Docker 볼륨을 통한 데이터 영속성

### 🖥️ 데스크톱 앱

#### 자동 설치
```bash
./install.sh
```

한 줄 명령어로 모든 것을 설치:
- ✅ 필수 도구 확인
- ✅ 데이터베이스 설정
- ✅ 앱 빌드 및 패키징
- ✅ /Applications에 설치
- ✅ 자동 실행

#### 자동 서비스 시작
앱 실행 시 자동으로:
- MySQL Docker 컨테이너 시작
- Spring Boot 백엔드 시작
- 데이터베이스 마이그레이션
- API 서버 준비 (10-20초 소요)

#### 번들 구조
모든 것이 앱에 포함:
- Spring Boot JAR (81MB)
- Docker Compose 설정
- 데이터베이스 연결 정보

### 🛠️ 기술 스택

#### 백엔드
- Spring Boot 3.2.2
- MySQL 8.0
- Flyway (DB 마이그레이션)
- JPA/Hibernate
- iText (PDF 생성)
- Apache POI (Excel 생성)

#### 프론트엔드
- Electron 28.3.3
- React 18
- TypeScript
- Vite
- Recharts (차트)
- Tailwind CSS

#### 인프라
- Docker & Docker Compose
- Maven (백엔드 빌드)
- electron-builder (앱 패키징)

### 📦 설치 요구사항

- **macOS** 11.0 이상
- **Docker Desktop** 20.10+
- **Java** 21+
- **Node.js** 18+

### 📝 알려진 제한사항

- macOS만 지원 (Windows/Linux 미지원)
- 코드 서명 없음 (보안 경고 발생 가능)
- 단일 사용자 모드 (멀티 유저 미지원)
- 클라우드 동기화 미지원

### 🐛 버그 수정

해당 없음 (첫 릴리스)
