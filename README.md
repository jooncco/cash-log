# 💰 cash-log
> A simple personal finance application, powered by [AI-DLC(v0.1.3)](https://github.com/awslabs/aidlc-workflows).

## ✨ Key Features

### 1. 📅 수입/지출 내역 관리
- **수입/지출 기록**: 날짜, 금액, 카테고리, 메모를 포함한 간편한 데이터 입력.
- **카테고리 커스터마이징**: 식비, 교통비, 쇼핑 등 본인의 라이프스타일에 맞춘 분류 설정.
- **필터링 & 검색**: 기간별, 카테고리별로 과거 내역을 빠르게 조회.

### 2. 📊 데이터 시각화 (Insights)
- **통계 대시보드**: 월별 지출 추이를 차트로 한눈에 확인.
- **카테고리별 비중**: 어떤 곳에 가장 많은 돈을 썼는지 파이 차트로 분석.

### 3. 🎯 예산 설정 및 알림
- **월간 예산 설정**: 이번 달 사용할 목표 금액을 설정하고 잔액 확인.
- **소비 경고**: 예산의 일정 비율(예: 80%, 100%) 초과 시 시각적 알림 제공.

### 4. 💾 데이터 영속성
- **로컬 DB 저장**: 작성한 내역이 휘발되지 않도록 안전하게 데이터 저장.
- **보고서 생성 기능**: 기록된 데이터를 CSV, EXCEL, PDF 파일로 저장.
